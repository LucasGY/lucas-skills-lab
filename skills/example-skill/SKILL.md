# Example Skill

## When to use

Use this skill when the user wants a repeatable workflow that benefits from task-specific instructions.

## Workflow

1. Confirm the task scope and constraints.
2. Keep the main instructions short and procedural.
3. Put detailed material in `references/` if it is only needed sometimes.
4. Put deterministic or repetitive actions into `scripts/`.
